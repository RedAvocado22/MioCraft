# MioCraft

> A Vietnamese programming knowledge base — from student projects to production-ready systems.

## About

MioCraft is a personal knowledge site built to document what I've learned about writing software that actually works in production. The content follows a series called *Code Sống Sót* ("Survival Coding"), structured around a real hospital management system built with Java 17, Spring Boot 3, MySQL, and Redis — because real examples teach better than toy ones.

Written for junior developers and final-year students who know the syntax but haven't shipped to production yet.

---

A Vietnamese programming knowledge base written for junior developers and final-year students. Articles are organized into 12 parts across three topics:

- **System Design** — load balancing, caching, message queues, rate limiting, circuit breakers, API gateways, distributed tracing, and more
- **Architecture** — clean code, SOLID principles, design patterns, layered architecture, and real-world case studies
- **Programming** — backend internals, database design, Spring Boot production patterns, deployment, and observability

🔗 **[mio-craft.vercel.app](https://mio-craft.vercel.app)**

## Stack

- [Astro 5](https://astro.build/) — static site generation
- [Vercel](https://vercel.com/) — deployment
- Vanilla CSS, JetBrains Mono, dark theme

## Local Development

```bash
git clone https://github.com/RedAvocado22/MioCraft.git
cd MioCraft
npm install
npm run dev
```
