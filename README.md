# MioCraft

A personal knowledge base about software development, written in Vietnamese.

🔗 **[mio-craft.vercel.app](https://mio-craft.vercel.app)**

## Stack

- [Astro 5](https://astro.build/) — static site generation
- [Vercel](https://vercel.com/) — deployment
- Vanilla CSS, JetBrains Mono, dark theme

## Local Development

```bash
git clone https://github.com/RedAvocado22/MioCraft.git
cd MioCraft
npm install
npm run dev
```
